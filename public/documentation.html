Movie API Documentation

movies_api is a Rest APi constructed in NODE.js to provide infofrmation to the user upon request to the server. The program is written in js, json, and html. The aim of this achievemnet is to learn how to use node on serverside development, and to get practive using the terminal-->

as a server side project, user experience will not be foregrounded, though the aim is of course to be as user friendly and accessible as possible. the bigger aim is to make the js files as readable and the modules as reusable as possible, so that I and other developers can understand and reuse the code.-->

  1. Return a list of ALL movies to the user
  2. Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  3. Return list of all Users
  4. Return data about a single user by Username
  5. Return data about a genre (description) by name/title (e.g., “Thriller”)
  6. Return data about a director (bio, birth year, death year) by name
  7. Allow new users to register
  8. Allow users to update their user info (username)
  9. Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later)
  10. Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later)
  11. Allow existing users to deregister (showing only a text that a user email has been removed—more on this later)
